What is Chronologic?

Current blockchains lack crucial time-based functionality such as transaction scheduling.

ChronoLogic supports temporal innovation with its partnership with the Ethereum Alarm Clock aiming to solve core problems on Ethereum.

WHAT IS DECENTRALIZED SCHEDULING?

The decentralized scheduling protocol executes scheduled transactions via a network of off-chain clients known as TimeNodes, which are incentivized to operate by a user-set bounty payment.

Users never need know about the smart contracts and TimeNodes - they just pay a small premium to schedule a transaction via wallets such as MyCrypto.

The TimeNode network consists of off-chain execution agents from the ChronoLogic community.

Why Time?

ChronoLogic's core Chronos dApp expands the possibilities of using time and adding unique features into smart contracts.

Ethereum was not originally designed for these exact use-cases — for example, automated sending in the future is not possible without a server-based solution. The Ethereum Alarm Clock partnership is aiming to solve this and other challenges.
